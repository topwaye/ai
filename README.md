# Ai

Use CodeArts Agent to create/open a project and enter prompts.

Open the SQL script file with MySQL Workbench 8.0 CE and execute it.

Use Eclipse IDE for Enterprise Java and Web Developers to open the project, and run the back end directory as Spring Boot App.

Copy the front end files to the Apache htdocs directory, and use a browser to visit the following URL address:

http://localhost
