## 🎯 核心功能

### 1. 引导加载程序 (boot.asm)
- ✅ 从软盘启动
- ✅ 加载内核到 0x8000
- ✅ 显示启动信息

### 2. 操作系统内核 (kernel.asm)
- ✅ 初始化系统环境
- ✅ 设置中断向量表
- ✅ 初始化设备驱动
- ✅ 主事件循环

### 3. 中断处理 (interrupts.asm)
- ✅ 0x60 中断处理程序
- ✅ 读/写操作完成处理
- ✅ EOI 信号发送
- ✅ 寄存器上下文管理

### 4. 设备驱动 (device_driver.asm)
- ✅ 设备初始化
- ✅ 读操作接口
- ✅ 写操作接口
- ✅ 状态查询
- ✅ 错误处理

### 5. 测试程序 (test.asm)
- ✅ 设备状态测试
- ✅ 写数据测试
- ✅ 读数据测试
- ✅ 结果显示

## 🔌 硬件接口

### 端口定义
| 端口地址 | 功能 | 方向 |
|---------|------|------|
| 0x60 | 状态寄存器 | 输入 |
| 0x61 | 数据端口 | 双向 |
| 0x62 | 控制寄存器 | 输出 |

### 控制命令
- `0x01` - 读命令
- `0x02` - 写命令
- `0xFF` - 复位命令

### 状态标志
- `bit 0` - 读数据就绪
- `bit 1` - 写数据就绪
- `bit 6` - 错误标志
- `bit 7` - 设备忙标志

## 🚀 快速命令

### 编译
```bash
# Windows
build.bat          # 编译操作系统
build_test.bat     # 编译测试程序

# Linux
make all           # 编译所有
make test          # 编译测试
```

### 运行
```bash
# 运行操作系统
qemu-system-x86_64 -fda os.img

# 运行测试
qemu-system-x86_64 -fda test.img

# 运行模拟器
python3 device_simulator.py
```

### 清理
```bash
# Windows
clean.bat

# Linux
make clean
```

## 📊 系统流程

### 启动流程
```
BIOS → 引导程序 → 内核 → 初始化 → 主循环
```

### 中断处理流程
```
设备中断 → CPU响应 → 保存上下文 → 处理程序 → 发送EOI → 恢复上下文 → 返回
```

### 设备操作流程
```
应用程序 → 设备驱动 → 硬件设备 → 中断通知 → 中断处理 → 完成回调
```
